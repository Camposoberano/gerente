═══════════════════════════════════════════════════════════════
    🔧 CONFIGURAÇÃO DO PLUGIN AMP
═══════════════════════════════════════════════════════════════

PASSO 1: Verificar Plugin AMP
──────────────────────────────────────────────────────────────

1. WordPress → Plugins
2. Verificar se "AMP" está instalado e ativo
3. Se não estiver:
   - Plugins → Adicionar Novo
   - Buscar: "AMP"
   - Instalar "AMP" by AMP Project Contributors
   - Ativar

PASSO 2: Configurar Modo AMP
──────────────────────────────────────────────────────────────

1. WordPress → AMP → Settings
2. Escolher modo: "Paired" (RECOMENDADO)

   ┌─────────────────────────────────────┐
   │ ◉ Paired                            │
   │   Usa URLs separadas (/amp/)        │
   │   Mantém site normal funcionando    │
   │   RECOMENDADO para News Soberano    │
   │                                     │
   │ ○ Transitional                      │
   │ ○ Standard                          │
   └─────────────────────────────────────┘

3. Salvar alterações

PASSO 3: Configurar Tipos de Conteúdo
──────────────────────────────────────────────────────────────

1. Aba "Supported Templates"
2. Marcar:
   ☑ Posts
   ☑ Pages
   ☑ Categories
   ☑ Tags
   ☑ Authors
   ☑ Archives

3. Salvar

PASSO 4: Configurar Analytics (Opcional)
──────────────────────────────────────────────────────────────

1. Aba "Analytics"
2. Adicionar:
   - Type: Google Analytics
   - Account ID: UA-XXXXXXXX-X ou G-XXXXXXXXXX

3. Salvar

═══════════════════════════════════════════════════════════════
    🔧 CONFIGURAÇÃO WP ROCKET PARA AMP
═══════════════════════════════════════════════════════════════

IMPORTANTE: WP Rocket não deve processar páginas AMP!

1. WP Rocket → Configurações
2. Aba "Avançado"
3. Seção "URLs to not cache"
4. Adicionar:
   /amp/
   ?amp=1
   ?amp
   /*/amp/

5. Aba "Otimização de Arquivos"
6. DESMARCAR para AMP:
   ☐ Minify CSS files
   ☐ Minify JavaScript files
   ☐ Combine JavaScript files (AMP)

7. Salvar e limpar cache

Se ainda houver erros, verificar:

═══════════════════════════════════════════════════════════════
    🆘 SOLUÇÃO DE PROBLEMAS
═══════════════════════════════════════════════════════════════

PROBLEMA: AMP não funciona
SOLUÇÃO:
   1. Verificar plugin AMP ativo
   2. Limpar cache WordPress
   3. Limpar cache WP Rocket
   4. Desativar outros plugins temporariamente
   5. Testar com tema v1.3.2

PROBLEMA: Muitos erros de validação
SOLUÇÃO:
   1. Instalar tema v1.3.2 (correções automáticas)
   2. Configurar WP Rocket corretamente
   3. Desabilitar plugins conflitantes
   4. Usar modo "Paired" (não Standard)

PROBLEMA: CSS syntax error
SOLUÇÃO:
   1. Tema v1.3.2 reduz CSS inline
   2. Verificar plugins adicionando CSS
   3. Usar Paired mode (não injeta todo CSS)
