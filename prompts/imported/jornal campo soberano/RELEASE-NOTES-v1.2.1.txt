═══════════════════════════════════════════════════════════════
    🎨 EXEMPLO DE USO
═══════════════════════════════════════════════════════════════

DESENVOLVEDORES/USUÁRIOS AVANÇADOS:

Você pode usar a nova função em qualquer lugar do tema:

<?php
// Exibir imagem ou placeholder
echo news_soberano_get_thumbnail('news-medium');

// Tamanhos disponíveis:
// - 'news-featured' (1200x600)
// - 'news-large' (800x450)
// - 'news-medium' (400x225)
// - 'news-thumb' (200x150)

// Específico de um post
echo news_soberano_get_thumbnail('news-large', $post_id);
?>

═══════════════════════════════════════════════════════════════
    ⚡ PERFORMANCE
═══════════════════════════════════════════════════════════════

IMPACTO ZERO NA PERFORMANCE:

   • SVG inline (~200 bytes vs ~50KB de imagem)
   • Sem requisições HTTP extras
   • Sem JavaScript necessário
   • Cache-friendly
   • Não aumenta tempo de carregamento

COMPARAÇÃO:
   Imagem JPEG 400x225:  ~50-150 KB
   Placeholder SVG:      ~0.2 KB

   ECONOMIA: 99.6% de bandwidth por placeholder
