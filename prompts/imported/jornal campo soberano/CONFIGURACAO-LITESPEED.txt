═══════════════════════════════════════════════════════════════
    🆘 SOLUÇÃO DE PROBLEMAS
═══════════════════════════════════════════════════════════════

PROBLEMA: Site quebrado após ativar CSS Combine
SOLUÇÃO:
   1. Page Optimization → CSS Combine: OFF
   2. Purge All
   3. Testar CSS Minify isolado
   4. Se funcionar, testar Combine novamente

PROBLEMA: Imagens não aparecem (lazy load)
SOLUÇÃO:
   1. Media → Lazy Load Images
   2. Exclude Classes: Adicionar "wp-post-image"
   3. Purge All
   4. Tema v1.3.3 já tem exclusões

PROBLEMA: JS não funciona
SOLUÇÃO:
   1. JS Defer: OFF temporariamente
   2. Identificar script problemático
   3. Adicionar em JS Excludes
   4. Reativar Defer

PROBLEMA: AMP com erros
SOLUÇÃO:
   1. Verificar exclusões AMP configuradas
   2. Tema v1.3.3 desabilita LiteSpeed no AMP
   3. Purge All
   4. Testar com ?amp=1#development=1

PROBLEMA: Cache não está funcionando
SOLUÇÃO:
   1. Verificar .htaccess tem regras LiteSpeed
   2. Server deve ser LiteSpeed (não Apache/Nginx)
   3. Plugin LiteSpeed ativo
   4. Cache: ON
