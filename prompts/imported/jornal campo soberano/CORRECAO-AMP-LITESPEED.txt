═══════════════════════════════════════════════════════════════
    🔧 CONFIGURAÇÃO OBRIGATÓRIA NO LITESPEED CACHE
═══════════════════════════════════════════════════════════════

PASSO 1: Exclusões de URLs
──────────────────────────────────────────────────────────────

1. WordPress → LiteSpeed Cache → Cache
2. Aba "Excludes"
3. Seção "Do Not Cache URIs"
4. Adicionar (uma linha por exclusão):

   /amp/
   /amp
   /*/amp/
   /*/amp

5. Seção "Do Not Cache Query Strings"
6. Adicionar:

   amp
   amp=1

7. Salvar alterações

PASSO 2: Exclusões de Otimização
──────────────────────────────────────────────────────────────

1. LiteSpeed Cache → Page Optimization
2. Aba "Tuning"
3. Seção "URI Excludes"
4. Adicionar:

   /amp/
   /amp
   /*/amp/
   /*/amp

5. Seção "JS Excludes"
6. Adicionar:

   /amp/

7. Seção "CSS Excludes"
8. Adicionar:

   /amp/

9. Salvar alterações

PASSO 3: Limpar Cache
──────────────────────────────────────────────────────────────

1. LiteSpeed Cache → Toolbox
2. Clicar "Purge All"
3. Aguardar confirmação
4. Testar AMP novamente

═══════════════════════════════════════════════════════════════
    🔍 DIAGNÓSTICO DE ERROS
═══════════════════════════════════════════════════════════════

ERRO: "type" is set to "litespeed/javascript"
──────────────────────────────────────────────────────────────
CAUSA: LiteSpeed está otimizando a página AMP
SOLUÇÃO:
   1. Verificar exclusões configuradas no plugin
   2. Purge All cache
   3. Testar novamente
   4. Tema v1.3.4 já desabilita via código

ERRO: "amp-custom" is missing in tag "style amp-custom"
──────────────────────────────────────────────────────────────
CAUSA: Múltiplas tags <style> sem atributo amp-custom
SOLUÇÃO:
   1. LiteSpeed combinando CSS incorretamente
   2. Desabilitar CSS Combine para AMP (já excluído)
   3. Tema v1.3.4 já desabilita via código

ERRO: "href" in "link rel=stylesheet" is invalid
──────────────────────────────────────────────────────────────
CAUSA: LiteSpeed gerando CSS combinado
SOLUÇÃO:
   1. Desabilitar CSS Combine para URLs AMP
   2. Adicionar /amp/ nas exclusões
   3. Tema v1.3.4 já desabilita via código

ERRO: Parent tag of "script" is "body", should be "head"
──────────────────────────────────────────────────────────────
CAUSA: LiteSpeed injetando scripts no rodapé
SOLUÇÃO:
   1. Desabilitar JS Defer para AMP
   2. Adicionar exclusões de JS
   3. Tema v1.3.4 já desabilita via código

ERRO: "target" is missing in tag "form"
──────────────────────────────────────────────────────────────
CAUSA: Forms sem atributo target="_top" ou "_blank"
SOLUÇÃO:
   1. Adicionar amp-form extension
   2. Adicionar target="_top" nos forms
   3. Será corrigido no próximo update
