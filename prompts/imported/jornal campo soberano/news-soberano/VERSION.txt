    IDENTIFICAÇÃO DA VERSÃO NO WORDPRESS
═══════════════════════════════════════════════════════════════

Para verificar a versão instalada:

1. PAINEL WORDPRESS:
   Aparência → Temas → News Soberano
   (Versão aparece na descrição)

2. CÓDIGO (functions.php):
   Constante: NEWS_SOBERANO_VERSION
   Valor: '1.2.1'

3. ARQUIVO style.css:
   Linha 7: Version: 1.2.1

═══════════════════════════════════════════════════════════════
    ATUALIZAÇÃO DO TEMA
═══════════════════════════════════════════════════════════════

Como atualizar de v1.1.0 para v1.2.0:

1. ✅ Fazer BACKUP do site atual
2. ✅ Fazer BACKUP do banco de dados
3. ✅ Desativar cache (WP Rocket)
4. ✅ Fazer upload do novo tema (substituir)
5. ✅ Ativar o tema atualizado
6. ✅ Verificar se a homepage mudou
7. ✅ Limpar cache
8. ✅ Testar responsividade
9. ✅ (Opcional) Marcar post como Hero (_is_ns_featured = 1)

IMPORTANTE:
- Totalmente compatível, sem perda de dados
- Configurações anteriores mantidas
- Posts e páginas não afetados
- Homepage agora usa template dedicado (front-page.php)
- Widgets preservados
