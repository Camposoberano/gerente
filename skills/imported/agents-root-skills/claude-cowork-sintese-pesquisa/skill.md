# 🎯 Codex Cowork - Síntese de Pesquisa para Tomada de Decisão

Não gaste horas lendo dezenas de páginas de documentos e dados brutos. Obtenha uma síntese analítica premium de no máximo meia página com recomendação embasada para tomar decisões confiáveis.

---

## 🎯 Resultado Esperado (Outcome-First)
- Processamento automático de todos os arquivos de pesquisa e documentação em uma única sessão.
- Síntese estruturada comparando Prós e Contras de todas as alternativas viáveis.
- Parecer e recomendação direta e altamente justificada para guiar o decisor.

## 🛠️ O Prompt de Execução

```markdown
Você é o Analista Estratégico e Tomador de Decisão do Cowork. Sua missão é digerir múltiplos arquivos de pesquisa e gerar um parecer executivo enxuto de no máximo meia página que nos ajude a tomar uma decisão estratégica.

Processo Analítico:
1. Leia e cruze todas as informações contidas na pasta de pesquisa fornecida.
2. Sintetize as vantagens (Prós) e desvantagens (Contras) de cada cenário/opção disponível.
3. Identifique onde estão os pontos cegos da nossa pesquisa (lacunas de dados que ainda precisamos obter).
4. Formule uma recomendação definitiva e clara justificando qual caminho é o mais racional de seguir.

**FONTES E LIMITES:**
- Pasta de documentos: [inserir caminho do diretório]
- O tema a decidir é: [ex: escolha do fornecedor de TI / aquisição de software CRM]
- Data limite para a decisão: [data]
- Salvar como: decisao_[tema]_[data].md

**ENTREGUE:**
1. Parecer Executivo estruturado e enxuto em formato Markdown.
2. Quadro comparativo visual das alternativas (Prós × Contras).
3. Lista de lacunas e recomendação final acionável.
```

## 💡 Estrutura da Resposta
- **Leitura Omnisciente**: O Cowork absorve todos os relatórios da pasta simultaneamente.
- **Síntese de Impacto**: Exposição clara do trade-off de cada alternativa.
- **Recomendação Definitiva**: Argumentação assertiva e orientada a dados para subsidiar o seu próximo passo.
